Exercício 2

Entrada e saída de dados

  • Escreva um programa que leia um número float e imprima seu valor


    Exemplo:


    Digite um número: 2

    O número digitado é: 2.000000

Solução

Exercicio 2

  • int main(){

    float n1;

    printf("Digite um número: ");

    scanf("%f", &n1);

    fflush(stdin);

    printf("O numero digitado e: %f", n1);

    return 0;

    }

Exercício 3

Entrada e saída de dados

  • Escreva um programa que leia um número inteiro e imprima o dobro desse valor.


    Exemplo:


    Digite um número: 2

    O dobro do seu número é: 4

Solução

Exercicio 3

  • int main(){

    int n1;

    printf("Digite um número: ");

    scanf("%i", &n1);

    fflush(stdin);

    printf("O numero %i x 2 = %i", n1, n1*2);

    return 0;

    }
