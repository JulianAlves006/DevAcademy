Exercício 1

Tipos de dados

  • Complete as declarações de variáveis com o tipo correto:

    a) ______ idade = 25;

    b) ______ nota = 8.75;

    c) ______ letra = 'A';

    d) ______ pi = 3.14159;

Solução

Exercício 1

Corrija os erros

Exercício 2

Exercício 3

Const e Escopo

  • Analise o código abaixo e responda:

    #include <stdio.h>
#define TAXA 0.1

int main() {
    const float desconto = 0.05;
    float preco = 200;
    float precoFinal;

    precoFinal = preco - (preco * desconto) + (preco * TAXA);
    printf("Preço final: %.2f", precoFinal);

    return 0;
}

    a) Qual o valor final impresso?

    b) O que aconteceria se removêssemos a linha #define TAXA 0.1?

    c) Qual a principal diferença entre usar #define e const nesse contexto?

Solução

Exercício 3

  • a) Resultado: 210.00 → 200 - (200 * 0.05) + (200 * 0.1) = 190 + 20 = 210

    b) O compilador acusaria erro, pois TAXA estaria indefinida.

    c) #define é um pré-processador textual e não respeita escopo.
    const é mais seguro, respeita o escopo e permite controle de tipo.
