Aula 3

Tipos de Dados e Variáveis

Esta aula tem como objetivo apresentar os principais tipos de dados e a declaração de variáveis na linguagem C.

1. Tipos Primitivos

  • int: números inteiros.
  • float: números decimais com menor precisão (até 7 casas).
  • double: números decimais com maior precisão (até 15 casas).
  • char: caracteres individuais.

2. Declaração e Inicialização

Para declarar uma variável, informe o tipo e o nome. Para inicializar, atribua um valor. 


int idade = 25;
char letra = 'A';
float altura = 1.75;

3. Constantes

Constantes são valores que não mudam durante a execução do programa.

  • #define PI 3.14 // Constante simbólica
  • const float PI = 3.14; // Constante com tipo

4. Tipos de Constantes

  • Inteira: printf("Idade: %d", 18);
  • Real: printf("PI: %.2f", 3.14);
  • Caractere: printf("Letra: %c", 'A');
  • String: printf("Mensagem: %s", "Olá");

5. Regras e Boas Práticas de Nomenclatura

  • Começar com letra ou underline.
  • Evitar palavras reservadas.
  • Seguir padrão camelCase ou snake_case.
  • Usar nomes significativos: mediaAluno, totalVendas.

6. Exemplos de Código

Exemplo 1: Declaração e Saída


#include <stdio.h>
  int main() {
  int idade = 20;
  printf("Idade: %d\n", idade);
  return 0;
}

Exemplo 2: Constante e Cálculo de área


#include <stdio.h>
int main() {
  const float PI = 3.14;
  float raio = 2.0;
  float area = PI * raio * raio;
  printf("Area: %.2f\n", area);
  return 0;
}

Exemplo 3: Entrada de Dados


#include <stdio.h>
int main() {
  float altura;
  printf("Digite sua altura: ");
  scanf("%f", &altura);
  printf("Altura: %.2f\n", altura);
  return 0;
}
