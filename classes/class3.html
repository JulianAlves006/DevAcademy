1. Introdução

O objetivo deste projeto é desenvolver um site educativo completo voltado para o ensino de lógica e programação básica na linguagem C, visando repassar os conhecimentos adquiridos durante o semestre para a sociedade, aprimorando a capacidade de pesquisa e organização da informação, além de incentivar o trabalho em equipe e a busca de conhecimento.

O site será estruturado de forma lógica e acessível, garantindo que os conteúdos sejam apresentados em uma ordem coerente para facilitar o aprendizado.

2. Estrutura do Conteúdo

2.1. Tipos de Dados e Variáveis

Tipos Primitivos

  • int: utilizado para armazenar números inteiros.
  • float: utilizado para armazenar números decimais com até 7 casas decimais.
  • double: semelhante ao float, mas com maior precisão (até 15 casas decimais).
  • char: utilizado para armazenar caracteres individuais.

Declaração e Inicialização de Variáveis

Em C, a declaração informa ao compilador o nome e tipo da variável. Para inicializá-la, atribuímos um valor.

Regras importantes:

  • Declarar antes de usar;
  • Nome não pode começar com número nem ser palavra reservada;
  • Finalizar com ponto e vírgula (;).

Exemplo de declaração:


int idade, telefone;
char letra, vogal;
float altura, media;

Exemplo de inicialização:


int idade = 25;
char vogal = 'A';
float altura = 1.75;

Constantes

Constantes são valores fixos durante a execução. Oferecem segurança, reutilização e facilidade de manutenção.

Tipos de constantes:

  • Inteira: printf("A idade mínima é %d\n", 18);
  • Real: printf("PI vale aproximadamente %.2f\n", 3.14);
  • Caractere: printf("Primeira letra: %c\n", 'A');
  • String: printf("Mensagem: %s\n", "Olá, mundo!");
  • Simbólica: #define PI 3.14159 ou const float PI = 3.14159;

Regras de Nomenclatura para Variáveis

Regras obrigatórias:

  • Começar com letra ou underline (_);
  • Pode conter letras, números e underlines;
  • Não usar palavras-chave (ex. int, return);
  • Sensível a maiúsculas/minúsculas (ex. valor ≠ Valor);

Boas práticas:

  • Usar nomes significativos (ex. mediaAluno);
  • Escolher um padrão: camelCase ou snake_case;
  • Variáveis booleanas com is, has, can (ex. isAtivo);

Exemplos corretos:


int contador;
float temperatura_media;
char inicial_usuario;

Exemplos incorretos:


int 123idade;        // Inicia com número
float valor total;   // Contém espaço
char for;            // Palavra reservada

3. Exemplos de Códigos Resolvidos

Exemplo 1: Cálculo da média de duas notas

Neste exemplo, usamos o tipo float para armazenar números decimais e calcular a média de duas notas. 


#include <stdio.h>

int main() {
    float nota1 = 7.5;
    float nota2 = 8.3;
    float media;

    media = (nota1 + nota2) / 2;

    printf("A média do aluno é: %.2f\n", media);

    return 0;
}

Saída esperada: A média do aluno é: 7.90

Exemplo 2: Uso de constantes e cálculo de área

Utilizamos uma constante para armazenar o valor de PI e calcular a área de um círculo. O uso de const garante que o valor não será alterado. 


#include <stdio.h>

int main() {
    const float PI = 3.14159;
    float raio = 5.0;
    float area;

    area = PI * raio * raio;

    printf("Área do círculo: %.2f\n", area);

    return 0;
}

Saída esperada: Área do círculo: 78.54

Exemplo 3: Entrada e exibição de informações do usuário

Neste exemplo, mostramos como declarar variáveis, usar scanf para receber dados do usuário e printf para exibir uma mensagem. 


#include <stdio.h>

int main() {
    int idade;
    char inicial;

    printf("Digite sua idade: ");
    scanf("%d", &idade);

    printf("Digite a inicial do seu nome: ");
    scanf(" %c", &inicial);

    printf("Você tem %d anos\n", idade);
    printf("E seu nome começa com '%c'\n", inicial);

    return 0;
}

Saída esperada (exemplo): Você tem 20 anos e seu nome começa com 'J'
