Printf() - Saída de dados em C

A função printf é utilizada para exibir informações na tela. Ela pertence à biblioteca padrão stdio.h.

Sintaxe:

printf("texto");

printf("Valor: %i", variavel);

Exemplo:

#include stdio.h

int main() {

printf("Hello, World!\n");

return 0;

}


Especificadores de formato comuns:

  • %i - inteiro - Ex: 1, 2, 3...
  • %f - ponto flutuante (float/double) - Ex: 1.5, 2.72 ...
  • %c - caractere - Ex: a, b, c...

Exemplo:

int idade = 20;

printf("A idade é %i anos\n", idade);

Saída: A idade é 20 anos


Formatação do float:

Com o %f (float), a saída de dados normalmente irá sair com diversos números decimais, mesmo sendo salvo somente um número inteiro.

Exemplo:

float n1 = 1;

printf("%f/n", n1);

Saída: 1.00

Scanf() - Entrada de dados em C

A função scanf é usada para ler dados digitados pelo usuário.

Sintaxe:

scanf("%i", &variavel);

Importante:

O operador & (e-comercial) é utilizado para passar o endereço da variável, permitindo que o valor digitado seja armazenado nela.

Exemplo:

#include stdio.h

int main(){

int idade;

printf("Digite sua idade: ");

scanf("%i", &idade);

fflush(stdin);

printf("Voce tem %i anos", idade);

}

Obs: Utilize o "fflush(stdin)" para poder retirar dados de espaços e enters desnecessários. Dessa forma o scanf só irá pegar as informações digitadas de fato.
