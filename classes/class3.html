Printf() - Saída de dados em C

A função printf é utilizada para exibir informações na tela. Ela pertence à biblioteca padrão stdio.h.

Sintaxe:

printf("texto");

printf("Valor: %i", variavel);

Exemplo:

#include stdio.h

int main() {

printf("Hello, World!\n");

return 0;

}

Especificadores de formato comuns:

  • %i - inteiro - Ex: 1, 2, 3...
  • %f - ponto flutuante (float/double) - Ex: 1.5, 2.72 ...
  • %c - caractere - Ex: a, b, c...

Exemplo:

int idade = 20;

printf("A idade é %i anos\n", idade);

Saída: A idade é 20 anos
