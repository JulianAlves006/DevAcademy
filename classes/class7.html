Estruturas de Repetição:

As estruturas de repetição, executam a repetição de um conjunto de instruções enquanto uma determinada condição é verdadeira.
Na linguagem C, existem três estruturas de repetição, são elas: while, do… while, for.
Cada uma dessas estruturas tem suas funções em termos de funcionamento. Todas serão apresentadas aqui, começando pelo While:

While():

Imagine que você tem uma jarra vazia e queira enchê-la de água, você só vai parar de
colocar água quando ela estiver cheia.
Na linguagem C, a função while é como uma instrução que diz: ”Continue fazendo isso
enquanto uma condição seja verdadeira”. Ou seja, o código vai continuar rodando
até você executar a opção que você escolheu para encerrar o programa.

Aqui está um exemplo prático de while():

#include <stdio.h>
                
int main() {
    int contador = 1;  // Inicializa o contador
                    
    while (contador <= 5) {  // Enquanto o contador for menor ou igual a 5
        printf("Contador: %i\n", contador);  // Imprime o valor do contador
        contador++;  // Incrementa o contador
    }
                    
    return 0;
}

Explicação:

  1. Incialiação: contador começa com valor 1.
  2. Condição: O loop while() executa enquanto contador <= 5.
  3. Corpo do loop:
    • Imprime um valor atual do contador
    • incrementa o contador com contador++.
  4. Término: Quando o contador chegar a 6, a condição contador <= 5 se torna falsa, e o loop termina.

Esse é um exemplo clássico de um loop whlle() para contar de 1 a 5.

do... while():

O do...while é um loop (laço de repetição) que garante que o bloco de código seja
executado pelo menos uma vez, mesmo que a condição de parada seja falsa desde o início.
Depois da primeira execução, ele verifica a condição para decidir se repete ou não.
Use quando tiver que fazer um loop que se execute ao menos uma vez.

Comparação com o while tradicional:

While do... while
Verifica a condição antes de executar. Executa primeiro e depois verifica a condição.
Pode nunca executar se a condição for falsa. Sempre executa pelo menos uma vez,
mesmo se a condição for falsa.

Exemplo prático: Validação de senha

#include <stdio.h>
int main() {
    int senha;
                    
    do {
        printf("Digite a senha (1234 para acertar): ");
        scanf("%i", &senha);
    } while (senha != 1234); // Repete enquanto a senha estiver errada
                    
    printf("Acesso permitido!\n");
    return 0;
}

Como funciona:

  1. O programa pede a senha pelo menos uma vez (mesmo que o usuário digite 1234 de primeira).
  2. Se a senha estiver errada (!= 1234), o loop repete.
  3. Quando a senha estiver correta, o loop termina.

for()

O for é mais um loop, usado quando você sabe quantas vezes um bloco de código deve ser executado.
Ele é composto por três partes principais:

  1. Inicialização
  2. Condição de continuação,
  3. Incremento/Atualização.

Sintaxe Básica:

for (inicialização; condição; incremento i++) {
                            //Bloco de código a ser repetido
                        }
Parte Descrição Exemplo
Inicialização Executa uma vez no início.
Define a variável de controle 		int i = 0;
Condição Verificada antes de cada interação.
Se falsa, o loop para. 		i < 10;
Incremento Executada após cada interação.
Atualiza a variável de controle. 		i++ ou i += 2;

Exemplo prático:

#include <stdio.h>
                                        
int main() {
    for (int i = 1; i <= 5; i++) {
        printf("%i ", i);
    }
    return 0;
}

Saída: 1 2 3 4 5

  1. i = 1 (inicialização).
  2. Verifica i <= 5 (verdadeiro).
  3. Executa printf("%d ", i).
  4. i++ (incrementa para 2).
  5. Repete até i ser 6 (quando a condição i <= 5 se torna falsa).
