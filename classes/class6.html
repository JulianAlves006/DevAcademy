Estruturas Condicionais

If e Else


A estrutura if é usada para verificar uma condição.

Se a condição for verdadeira, o código dentro do bloco if será executado.

Se for falsa, o código dentro do bloco else será executado (se existir).


Por exemplo:

Explicacao:

Se a variável idade for maior ou igual a 18, o programa imprime "Você é maior de idade!".

Caso contrário, imprime "Você é menor de idade!".

Estrutura if-else if-else


Às vezes, precisamos verificar várias condições.

Para isso, usamos if, seguido de else if para checar outras condições, e finalmente else para o caso em que nenhuma das condições anteriores é verdadeira.

Por exemplo:

Explicacao:

Se a nota for maior ou igual a 9, imprime "Aprovado com distinção!".

Se não, mas for maior ou igual a 6, imprime "Aprovado!".

Se for menor que 6, imprime "Reprovado!".

Estrutura switch e case


O switch é utilizado quando temos várias opções para verificar, tornando o código mais organizado quando há muitas condições a serem verificadas.

Por exemplo:

Explicacao:

O valor de dia é comparado com os casos dentro do switch.

Se dia for 3, imprime "Terça-feira".

Se for menor que 6, imprime "Reprovado!".Se o valor não corresponder a nenhum case, o programa vai para o default, que imprime "Dia inválido".

Uso do break e default


Break: Serve para sair do bloco do switch ou de um laço (for, while).

Default: Serve para quando nenhum dos case do switch for verdadeiro, ele executa o código do default.

No exemplo do switch, o break faz com que o programa saia da estrutura switch assim que um dos casos for encontrado.
