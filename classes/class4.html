Operadores e Expressões

Na linguagem C, operadores são símbolos usados para realizar operações com variáveis e valores. Expressões combinam operadores com operandos (variáveis ou valores) para produzir um resultado.

Imagine que o computador é como uma calculadora super poderosa. Quando você quer fazer contas ou tomar decisões com ele, você usa expressões, que são como pequenas frases que dizem o que você quer calcular ou comparar.

Para montar essas "frases", usamos operadores.


O que é um operador?


Um operador é um símbolo que diz ao computador o que fazer com valores ou variáveis (como números guardados na memória).

Por exemplo:

  • + é o operador de adição (soma).
  • > é o operador de maior que (comparação).
  • && é o operador de "e" lógico (ver se duas coisas são verdadeiras).

O que é uma expressão?

Uma expressão é uma combinação de valores e operadores

Por exemplo:

3 + 5
É uma expressão aritmética. Ela soma 3 e 5. Resultado: 8.

Tipos de Operadores

1. Operadores Aritméticos (matemática básica)

Símbolo Nome O que faz Exemplo
+ Soma Soma dois números 3 + 2 = 5
- Subtração Subtrai um número do outro 5 - 2 = 3
* Multiplicação Multiplica dois números 4 * 3 = 12
/ Divisão Divide um número pelo outro 10 / 2 = 5
% Resto Mostra o resto da divisão inteira 7 % 3 = 1

* O operador % serve pra ver o resto. Por exemplo: 7 balas divididas para 3 crianças dá 2 balas cada, e sobra 1 → esse 1 é o resto.

2. Operadores Relacionais (comparações)

Esses operadores comparam dois valores e dizem se a comparação é verdadeira (1) ou falsa (0)

Símbolo Significado Exemplo Resultado
== Igual a 5 == 5 Verdadeiro (1)
!= Diferente de 5 != 3 Verdadeiro (1)
< Menor que 3 < 5 Verdadeiro (1)
> Maior que 10 > 6 Verdadeiro (1)
<= Menor ou igua 4 <= 4 Verdadeiro (1)
>= Maior ou igual 5 >= 2 Verdadeiro (1)

3. Operadores Lógicos (usados em decisões)

Esses operadores combinam comparações. Muito usados em "se isso e aquilo" ou "se isso ou aquilo".

Símbolo Nome O que faz Exemplo
&& E (AND) Verdadeiro só se as duas condições forem verdadeiras (idade > 18 && idade < 60)
|| OU (OR) Dá verdadeiro se uma só for verdadeira (idade < 18 || idade > 60)
! NÃO (NOT) Inverte o valor: se era verdadeiro, vira falso !(x > 0) → se x for 1, vira 0
Ordem de Precedência dos Operadores

Quando uma expressão tem vários operadores misturados (como +, *, &&, ==...), o C precisa saber qual parte resolver primeiro.

A ordem de precedência é a regra que define quais operadores vêm primeiro na hora de calcular uma expressão.

  1. () – Parênteses
  2. !, unário - – NOT, negativo
  3. *, /, % – Multiplicação, divisão, módulo
  4. +, - – Adição, subtração
  5. <, <=, >, >=
  6. ==, !=
  7. &&
  8. ||
4. Exemplo prático:

Exemplo 1: Operadores Aritméticos

#include <stdio.h>

int main() {

    int a = 10, b = 3;

    printf("Soma: %d\n", a + b);

    printf("Subtração: %d\n", a - b);

    printf("Multiplicação: %d\n", a * b);

    printf("Divisão: %d\n", a / b);

    printf("Resto: %d\n", a % b);

    return 0;

}

Esse código faz operações matemáticas básicas com dois números (a = 10 e b = 3) e mostra os resultados

Exemplo 2: Relacionais e Lógicos

#include <stdio.h>

int main() {

    int idade = 20;

    if (idade >= 18 && idade <= 60) {

        printf("Adulto.\n");

    } else {

        printf("Fora da faixa adulta.\n");

    }

    return 0;

}

Esse código verifica se uma idade está entre 18 e 60 anos

Exemplo 3: Precedência

#include <stdio.h>

int main() {

    int resultado = 10 + 5 * 2;

    printf("Resultado: %d\n", resultado); // Saída: 20 (multiplica antes)

    return 0;

}

Esse código mostra como o C decide qual operação fazer primeiro numa expressão.

Dica Final

Se você pensar nos operadores como símbolos que ligam pedaços de informação, eles vão ficar mais fáceis de entender.

  • Operadores que fazem contas → Aritméticos
  • Operadores que comparam valores → Relacionais
  • Operadores que juntam condições → Lógicos
